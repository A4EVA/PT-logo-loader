pub fn add_child<W: Widget<T> + Clone>(&mut self, child: W, flex: f64)[src]

Add a child widget.

If flex is zero, then the child is non-flex. It is given the same constraints on the "minor axis" as its parent, but unconstrained on the "major axis".

If flex is non-zero, then all the space left over after layout of the non-flex children is divided up, in proportion to the flex value, among the flex children.
